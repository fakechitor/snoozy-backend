# Project description
Backend part of Snoozy app.

Stack:
- Java
- Spring
- PostgreSQL
- Apache Kafka
- Minio
- Docker

Required .env file:
```
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=
MINIO_ROOT_USER=
MINIO_ROOT_PASSWORD=
OAUTH_WEB_CLIENT=
KAFKA_BOOTSTRAP_SERVERS=
JWT_SECRET_TOKEN=
AUTH_URI=
USER_URI=
GROUP_URI=
```
